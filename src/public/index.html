使用说明

  1. 添加自定义 API Key 和对应的 Cursor Cookie 值。
  2. API Key 可以不添加 Cookie，但不会有任何功能，可以后续再添加 Cookie。
  3. 使用自定义 API Key 作为 OpenAI API 的认证凭证。
  4. 系统将自动在多个 Cookie 之间进行轮询。
    • /v1/models - 模型列表
    • /v1/chat/completions - 聊天补全
